# # Bazar Ohkubo

Site oficial do Bazar Ohkubo desenvolvido para apresentar produtos, realizar pedidos e integrar pagamentos através do Mercado Pago.

## Funcionalidades

* Catálogo de produtos
* Carrinho de compras
* Gestão de pedidos
* Integração com Mercado Pago
* Design responsivo
* Painel administrativo

## Tecnologias

* HTML5
* CSS3
* JavaScript
* GitHub Pages
